# one-key-prompt

> 一个名字，比一万字描述更管用。

把冗长提示词炼成一个精准的名字、理论或概念。你不需要教 AI 怎么做，只需要告诉它「用谁的方法」。

---

### 118 字 → 18 字

**你的原版：**

> 帮我写一篇产品文案，要让人觉得这个产品很特别，不是那种烂大街的推销感，要有一种好像朋友在跟你聊天的感觉，看完之后很想买但又不觉得被推销了，语气要轻松但不是随便，要让人觉得这个品牌很懂自己。

**炼金版：**

> 用拉斯韦尔的「只对一个人说话」原则写产品文案。

118 字描述的感觉，一个名字就激活了。

### 102 字 → 15 字

**你的原版：**

> 我同事最近做了个很蠢的决定但我不好直说，我想劝他改主意但他这人特别固执不爱听别人的意见，你帮我想个说法，语气要温和但得让他意识到问题，不能伤他面子但是又得点中要害，最好让他自己得出结论。

**炼金版：**

> 用苏格拉底式提问法劝同事改主意。

100 字在纠结「怎么说」，15 字直接锁定了「用什么方法说」。

## 安装

```bash
claude mcp add-plugin https://github.com/sdyckjq-lab/one-key-prompt
```

装完直接用 `/one-key-prompt` 或 `/提示词炼金术`。也支持 Codex CLI、OpenClaw 等 AI 开发环境。

## 它能做什么

**正向炼金** — 贴一段冗长的提示词，自动诊断 → 推荐 3 个维度的激活钥匙（人物 / 理论 / 概念）→ 输出可直接复制的精简版。编程、写作、分析任务会自动调整推荐风格。

**反向炼金** — 已知一个概念名，想知道它适合什么场景？直接输入「费曼学习法怎么用」，获取适用场景、不适用场景和提示词模板。

**深度解读** — 对推荐的钥匙感兴趣？输入「解读 塔勒布」，查看核心要点、高级用法和组合配方（如「反脆弱 + 幂律分布 = 理解黑天鹅事件的完整框架」）。

**诊断：**

```
🔍 诊断结果

模糊度：8/10
效果预估：★★☆☆☆（用 100 多字描述一种感觉，没给方法）
核心意图：写一篇有说服力但不像推销的产品文案
主要问题：
  - 用 100 多字描述一种感觉，而不是给出方法
  - "朋友聊天的感觉""不觉得被推销"→ 这些是结果，不是指令
```

**炼金：**

```
✨ 炼金结果

👤 乔布斯 — 极致简化和用户体验驱动产品设计
  💡 为什么适合：你要的"不像推销"正是乔布斯产品叙事的标志
  📋 直接用：以乔布斯的产品叙事风格写{产品}文案
  🤔 也考虑了格拉德威尔，但他偏分析叙事，不如乔布斯匹配"产品+不推销"

📊 PAS 框架 — 先戳痛点，再搅动情绪，最后给方案
  💡 为什么适合：你要"看完想买但不觉得被推销"，PAS 天然带着情绪驱动
  📋 直接用：用 PAS 文案框架写{产品}，先描述痛点再搅动情绪最后自然引出产品

🔑 钩子-主体-号召 — 先抓注意力，再传递价值，最后推动行动
  💡 为什么适合：你缺的不是感觉，是一套把感觉落地成文案的方法论
  📋 直接用：关于{产品}，用钩子-主体-号召结构写文案
```

## 知识库

内置 75 个精选激活钥匙，覆盖人物（塔勒布、芒格、费曼、乔布斯...）、理论（波特五力、第一性原理、金字塔原理...）、概念（确认偏误、橡皮鸭调试、复利效应...）三大类。

支持自定义扩展，详见 [知识库文档](knowledge-base/)。
