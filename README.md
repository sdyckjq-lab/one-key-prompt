# 提示词炼金术

> 一个名字，比一万字描述更管用。

把冗长的提示词炼成精准的「概念激活钥匙」——用一个恰当的人名、理论或概念，比长篇大论更有效地激活 AI 的知识网络。

## 它解决什么问题

大多数人写提示词越写越长，试图用详细描述覆盖所有细节。但冗长的描述是在「教」AI，而不是「激活」AI 已有的知识网络。

**核心洞察**：告诉 AI「用塔勒布的反脆弱框架」，比花一千字描述「你要考虑从混乱中获益的能力」更有效。

## 效果对比

**你的原版（155 字）：**

> 我想让你帮我深入分析一下我们公司目前面临的竞争环境，要考虑同行竞争对手的定价策略、供应商的议价能力、新进入者可能带来的威胁、替代产品的风险，还有我们客户的选择余地。然后基于这些分析，给我一些具体的战略建议，告诉我们在哪些方面可以加大投入，哪些方面应该收缩。另外还要考虑一下整个行业未来的发展趋势，新技术可能带来的颠覆性影响。

**炼金版（45 字）：**

> 用波特五力模型分析我们行业的竞争格局，评估五力后给出战略建议。再用反脆弱框架检查策略韧性，用二阶效应预判对手反应。

短了 70%，效果好 3 倍。因为「波特五力」四个字就激活了 AI 对五种竞争力量的完整知识。

## 安装

```bash
# Claude Code 用户
claude mcp add-plugin https://github.com/sdyckjq-lab/one-key-prompt
```

安装后直接在对话中使用 `/提示词炼金术` 或 `/prompt-alchemy` 即可。

## 使用方法

### 基本用法

直接调用命令，贴上你的提示词：

```
/提示词炼金术
```

然后输入你想优化的提示词。Skill 会自动完成三步：

**第一步：诊断** — 分析你的提示词哪里写得模糊、哪里可以精炼

```
🔍 诊断结果

模糊度：7/10
核心意图：竞争环境分析与战略规划
主要问题：
  - 用了 100 多字描述一个现成框架能概括的事
  - 缺少分析框架，纯描述需求
```

**第二步：炼金** — 从三个维度推荐激活钥匙（人物 / 理论 / 概念）

```
✨ 炼金结果

👤 塔勒布 — 反脆弱，从混乱中获益
📊 波特五力模型 — 五种竞争力量分析行业格局
🔑 二阶效应 — 不只看直接后果，还要想后果的后果
```

每个钥匙都附带一句可以直接复制使用的提示词。

**第三步：对比** — 原版 vs 炼金版，一目了然

### 深度探索

对某个激活钥匙感兴趣？输入「解读 + 名称」深入理解：

```
解读 波特五力模型
```

Skill 会解释这个概念的内涵、核心要点、在提示词中的高级用法，以及相关的激活钥匙。

## 知识库

内置 75 个精选激活钥匙，覆盖：

| 类型 | 数量 | 示例 |
|------|------|------|
| 人物 | 25 | 塔勒布、芒格、费曼、孙子、乔布斯... |
| 理论 | 25 | 波特五力、第一性原理、金字塔原理、蓝海战略... |
| 概念 | 25 | 确认偏误、橡皮鸭调试、复利效应、逆向思维... |

### 自定义扩展

在项目目录下创建 `.prompt-alchemy/knowledge-base/` 文件夹，添加自己的激活钥匙：

```json
{
  "version": 1,
  "entries": [
    {
      "id": "differential-diagnosis",
      "name": "鉴别诊断思维",
      "core_idea": "系统排除法，从症状推导最可能的病因",
      "scenarios": ["分析", "学习"],
      "keywords": ["诊断", "排查", "鉴别", "排除法", "病因分析"],
      "prompt_template": "用鉴别诊断的思维框架分析{topic}，列出可能的原因并逐一排查",
      "popularity": "medium"
    }
  ]
}
```

详见 `.prompt-alchemy/knowledge-base/README.md`。

## 原理

灵感来自两个认知科学洞察：

1. **隐性知识理论**（Michael Polanyi）—— 我们知道的东西远多于能说出来的。AI 模型也是：它「知道」的东西远多于你能在提示词里描述的。精准的概念名称能激活这些隐性知识。

2. **参数空间的知识组织**（Ilya Sutskever）—— AI 模型的参数空间中，人名和概念标签对应着高度组织化的知识簇。提到「费曼」，AI 就自动激活了费曼学习法、科学直觉、反直觉思维等一整套知识网络。
